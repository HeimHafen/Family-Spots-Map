# Code of Conduct

Wir möchten, dass Family Spots Map ein freundlicher Ort für alle ist.

- Sei respektvoll und hilfsbereit.
- Kein Hass, keine Diskriminierung, keine Belästigung.
- Konstruktive Kritik ist willkommen, persönliche Angriffe nicht.

Verstöße können dazu führen, dass Beiträge entfernt oder Personen aus der
Community ausgeschlossen werden.
