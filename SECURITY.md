# Security Policy

## Supported Versions

Die Web-App ist ein statisches Projekt ohne Server-Backend. Sicherheitsrelevante
Änderungen werden im `main`-Branch veröffentlicht.

## Reporting a Vulnerability

Wenn du eine Sicherheitslücke findest, melde sie bitte vertraulich per E-Mail
an **[DEINE KONTAKTADRESSE]** und gib uns Zeit, das Problem zu analysieren und zu
beheben, bevor Details öffentlich werden.
